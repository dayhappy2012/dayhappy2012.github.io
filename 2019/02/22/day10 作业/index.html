  4. 实现属于自己的字典update方法：用一个字典去更新另一个字典的元素(不能使用自带的update方法)
    yt_update(字典1， 字典2)

  5. 实现属于自己的items方法：将字典转换成列表，字典中的键值对转换成元祖。（不能使用items方法）
    yt_items(字典)
    例如：{‘a’: 1, ‘b’:2, ‘c’:3} —> [(‘a’, 1), (‘b’, 2), (‘c’, 3)]
